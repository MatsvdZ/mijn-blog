# Meesterschap Blog

Een interactieve one-page website waarin ik terugblik op mijn leerproces tijdens de minor Web Design & Development.

## Over het project

Dit blog laat mijn ontwikkeling zien aan de hand van:

- Weekly Nerd sessies
- Sprint 0
- Hackathon
- Vakken
- Meesterproef
- Reflectiegesprekken

De website is opgebouwd als een interactieve ervaring met een game-geïnspireerde interface, animaties en een custom carousel.

## Gebruikte technieken

- HTML5
- CSS3
- JavaScript
- CSS Custom Properties
- Session Storage
- SVG animaties

## Functionaliteiten

- Interactieve carousel navigatie
- Keyboard navigatie
- Thema switcher (blauw / rood)
- Intro scherm
- Onthouden van actieve pagina
- Onthouden van carousel positie
- Diverse interactieve elementen en easter eggs

## Toegankelijkheid

Bij de ontwikkeling is aandacht besteed aan:

- Semantische HTML
- Keyboard navigatie
- Focus states
- ARIA labels
- Beschrijvende alt-teksten
- Voldoende kleurcontrast
